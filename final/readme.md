##Spring 2017

A science experiment showing the health benefits of alcoholic drinks taken before a meal to stimulate the appetite. This project focuses on the cocktail drink, Negroni.

Ingredients for Negroni:
* Gin
* Campari
* Sweet Vermouth

Equipment:
* 3 Solenoid Valves
* 3 LEDs
* 3 Diodes
* 3 Transistors
* 12V Battery
